using these script you can let a node server listen to messages it gets from visitors and make an arduino 
do something you want

add connect.js and node_modules/socket.io/node_modules/socket.io-client/socket.io.js to your index file
call connect(); in your index file

run main.js as a node in your terminal

make sure to change te IP adress in main.js to the IP your computer is

these scripts together will let a light on your arduino go on when someone is visiting your page.
you can write your own arduino code in main.js
